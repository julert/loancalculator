# Калькулятор займа

Простой и удобный калькулятор для расчета платежей по займу.

## Функциональность

- Расчет аннуитетных и дифференцированных платежей
- График погашения на первые 12 месяцев
- Визуализация переплаты по займу
- Адаптивный дизайн для мобильных устройств

## Как использовать

1. Введите сумму займа
2. Укажите срок в месяцах
3. Задайте процентную ставку
4. Выберите тип платежа
5. Нажмите "Рассчитать"

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
